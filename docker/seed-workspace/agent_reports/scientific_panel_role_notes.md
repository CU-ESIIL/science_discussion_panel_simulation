# Scientific Panel Role Notes

Use this file for role-specific operating notes that apply across the
Scientific Panel Digital Twin. Do not create hidden role templates outside the
shared architecture in `AGENTS.md`.

## Role Review Pattern

Each role review should include:

- role name;
- question or artifact reviewed;
- contribution summary;
- topic tags;
- stance;
- confidence;
- evidence references;
- unresolved questions;
- proposed decisions or norms;
- action items and owners.

The Discussion Intelligence Agent should convert role reviews into structured
events for dashboard use.

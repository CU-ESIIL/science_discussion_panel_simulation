# HEARTBEAT.md - Periodic Check Notes

Keep this file empty or comment-only unless the user explicitly asks for a recurring check.

Heartbeat requests should be specific, bounded, and reviewable. Do not use heartbeat tasks to bypass human review, run expensive jobs silently, send external messages, publish outputs, or modify credentials.

# Moderator And Support Roles

The Scientific Panel Digital Twin uses one visible Moderator, one Organizer,
six panelists, and one structured-record support agent.

| Agent | Responsibility |
| --- | --- |
| Cibele Amaral (Moderator) | Introduces topics, asks prepared and follow-up questions, keeps participation balanced, redirects unsupported claims, summarizes transitions, and closes each discussion. |
| Jennifer Balch (Organizer) | Selects themes, proposes questions, reviews final reports, aligns work with workshop goals, schedules sessions, and identifies external experts. Jennifer does not participate in the scientific debate. |
| Discussion Intelligence Agent | Records every meaningful contribution as structured metadata and tracks agreement, disagreement, assumptions, evidence gaps, future research priorities, related norms, and action items. |

Scientific reasoning comes from the six panelists: Tanya Berger-Wolf, Lauren
Gillespie, Jenna Kline, Justin Kitzes, Katherine Siegel, and Ty Tuff. Their
roles represent public scientific expertise and perspective only, not private
views or personal impersonation.

# Sample Promoted Report

This page demonstrates how a private workspace output can be promoted into the public MkDocs site.

## Source

- Original working location: `/workspace/documents/example_report.md`
- Review status before promotion: `approved`
- Promotion target: `docs/reports/sample-promoted-report.md`

## Summary

The public report contains only reviewed narrative, lightweight figures, and links or metadata for large outputs. Private drafts, prompt logs, unpublished data, and credentials remain outside the public site.

## Provenance Pattern

Every promoted report should identify:

- source workspace files
- scripts or notebooks used to produce outputs
- data sources and license constraints
- review status
- unresolved assumptions
- human approval record

# OASIS Scientific Discussion Panel

OASIS Scientific Discussion Panel is a persistent evidence-based scientific
forum for **AI for Ecology: Accelerating Discoveries, Reducing Uncertainties,
and Scaling Solutions**.

It is powered by OpenClaw and preserves the ScienceClaw infrastructure: Docker,
OpenClaw runtime, AI-VERDE model routing, mounted secrets, workspace
persistence, CMS/file review, GitHub repository manager, MkDocs, optional Slack,
and optional bounded workers.

The central product is the continuing discussion. Notes, transcripts, evidence
ledgers, experiments, summaries, and public reports support that discussion.

## Start

```bash
cp .env.example .env
make init-panel
make doctor
make demo
```

## Representation

The panelists are disclosed simulations informed by documented expertise and
source material. They do not speak for the real people whose work inspired the
perspectives, and generated statements must never be presented as private views.

# Dashboard Patterns

Dashboards in ScienceClaw should be publishable as static MkDocs pages whenever possible.

Recommended patterns:

- Static HTML reports generated by worker jobs.
- PNG or SVG figures with source scripts and metadata.
- Small CSV or JSON summaries committed under `docs/assets/data/`.
- Public STAC, COG, Zarr, or Parquet endpoints referenced by URL.
- Metadata-only summaries for private or restricted datasets.

Avoid dashboards that require private `/workspace` files, local credentials, or a running container to render on GitHub Pages.

See the [sample dashboard](dashboard/sample-storage-dashboard.md) for a minimal pattern.

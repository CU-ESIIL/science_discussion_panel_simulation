# Sample Storage Dashboard

This sample shows how a large external output can be represented publicly without committing the underlying data.

| Field | Example |
|---|---|
| Store | `public_stac_demo` |
| Data pattern | STAC metadata with streamable COG assets |
| Public endpoint | `https://earth-search.aws.element84.com/v1` |
| Private workspace source | `/workspace/analysis/example_stac_preview` |
| Published artifacts | small report, figure, metadata summary |
| Large data handling | linked or streamed, not committed |

Public dashboards should remain readable on GitHub Pages without private credentials.

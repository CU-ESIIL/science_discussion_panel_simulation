# Panel Architecture

OASIS Scientific Discussion Panel keeps the existing ScienceClaw/OpenClaw
infrastructure while changing the operating model.

```text
GitHub = control plane
repository = durable public configuration and memory
container = replaceable runtime
workspace = active panel memory
external storage = large or durable source material
```

The central product is continuing evidence-based discussion. Reports,
transcripts, notes, evidence tables, reading lists, and experiment summaries
support the discussion.

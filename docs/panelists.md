# Scientific Panel Avatars

The default Scientific Panel Digital Twin is a moderated panel about AI for
ecology. Named roles are disclosed avatars based on public online personas,
public scientific work, and publicly available professional context. They are
not personal impersonations, statements from the real people, endorsements, or
claims about private views.

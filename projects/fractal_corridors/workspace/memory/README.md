# Memory

Use this directory for structured memory files that support the working group.

Daily raw notes belong in `/workspace/daily_notes`. Promote durable project-level memory to `/workspace/MEMORY.md`.

## Quarantine

Use `memory/quarantine/<project_slug>/` when different projects, research questions, hypotheses, datasets, or audiences need isolated working memory.

See `documents/MEMORY_QUARANTINE_PROTOCOL.md` for the rules.

Suggested files inside each quarantine:

- `README.md`
- `working_memory.md`
- `assumptions.md`
- `decisions.md`
- `evidence.md`
- `handoff.md`

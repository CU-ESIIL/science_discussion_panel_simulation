# DECISION_PROTOCOL.md - How the Team Decides to Decide

Status: draft; not yet human-approved.
Date: 2026-05-17
Owner: PI Liaison / User Interview Agent

This protocol defines how the working group chooses a decision process before making substantive decisions.

## Decision classes

| Class | Examples | Default decision process | Required record |
| --- | --- | --- | --- |
| Routine coordination | File ownership, meeting notes, artifact routing | Owning role decides within its scope | Relevant working file |
| Scientific scope | Research question, phase exits, claim readiness | Scientific Director proposes; PI reviews when direction changes | `DECISIONS.md`, `PROJECT_CHARTER.md` |
| Methods and analysis | Data filters, model design, diagnostics, uncertainty treatment | Quantitative Modeler or Data Engineer proposes; Scientific Director reviews; Skeptic reviews before promotion | `ASSUMPTIONS.md`, analysis notes |
| Evidence and citation | Source inclusion, claim support, data licenses | Citation & Evidence Curator audits; Scientific Director resolves disputes | literature notes, evidence tables |
| Narrative promotion | Moving from exploratory notes to draft claims | Narrative Lead proposes; Citation Curator and Skeptic review | claim-evidence map, review memo |
| Sensitive or external action | Publishing, policy language, public health claims, legal claims, Indigenous or Tribal claims, external messages, credentials, deletions, expensive jobs | Human approval required before action | `HUMAN_REVIEW.md`, approval note |

## Decision rule

Before deciding, the owner should state:

1. The decision being made.
2. Why it is needed now.
3. Which decision class applies.
4. What evidence is available.
5. Who must review it.
6. Where the outcome will be recorded.

## Escalation

- If evidence is insufficient, record an assumption or question instead of deciding.
- If the decision affects project direction, route it to the PI through the PI Liaison.
- If the decision affects claims, require Citation Curator and Skeptic review before promotion.
- If the decision affects communities, public health, law, policy, sovereignty, publication, deletion, credentials, external communication, or cost, require explicit human approval.

## Default bias

When uncertain, the group should prefer reversible, documented, internally visible steps over irreversible, external, expensive, or public steps.

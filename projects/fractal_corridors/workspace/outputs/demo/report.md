# Demo Environmental Workflow

This report was generated by `scripts/demo_environmental_workflow.py`.

The workflow creates a synthetic 10 by 10 habitat-suitability grid from toy elevation, moisture, and disturbance gradients. It is designed to prove that the ScienceClaw appliance can run a reproducible scientific workflow, write outputs, and expose artifacts for review. It is not intended as a real ecological model.
